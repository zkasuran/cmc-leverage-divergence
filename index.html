CMC integration — CoinMarketCap 20 Index (CMC20)

CMC BSC  CMC's own index, on BNB Chain

CMC20 is CoinMarketCap's benchmark index — the top 20 cryptocurrencies by market cap (ex-stables, ex-wrapped), tokenized as a BEP-20 on BNB Smart Chain via Reserve Protocol. Contract 0x2f8A…6867. It is the natural investable benchmark for any CMC-data-driven strategy, and it only exists on BNB Chain.

CMC Data API — the source

The CMC20 price history in this repo is fetched from api.coinmarketcap.com/data-api/v3/cryptocurrency/historical?id=38442 — CMC's free public data API. No Binance. No third party. The funding-rate data that powers the strategy comes from the CMC AI Agent Hub MCP (get_global_crypto_derivatives_metrics). The closed loop: CMC data → CMC Agent Hub signal → BNB Chain execution → CMC20 benchmark.
